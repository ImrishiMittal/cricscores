---

## 📖 About

**CricScorers** is a modern cricket scoring and tournament management platform designed for local clubs, academies, corporate tournaments, and gully cricket.

It provides a complete ecosystem—from live ball-by-ball scoring to full tournament management—with professional scorecards, player analytics, public tournament sharing, and detailed statistics.

---

# ✨ Features

## 🏏 Live Match Scoring

* Ball-by-ball scoring
* Runs, wickets & extras
* Strike rotation
* Partnership tracking
* Fall of wickets
* Required Run Rate (RRR)
* Current Run Rate (CRR)
* Undo last ball
* Super Over support
* DLS (Duckworth–Lewis–Stern) support

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/ImrishiMittal/cricscores.git
```

```bash
cd cricscores
```

## Install Frontend

```bash
cd Frontend/vite-project
npm install
npm run dev
```

## Install Backend

```bash
cd Backend
npm install
npm run dev
```
